Thanks for your interest in contributing! Please follow these steps:

1. Fork the repository and create a feature branch from `main`.
2. Make small, focused commits and include tests for changes.
3. Run the test suite locally (`npx truffle test`) and ensure linting passes if added.
4. Open a pull request describing the change and linking any relevant issues.

Please use GitHub issues for bug reports or feature requests, and be courteous and professional in all interactions.
